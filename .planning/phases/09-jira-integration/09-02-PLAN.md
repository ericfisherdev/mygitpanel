**Struct and constructor:**
```go
type JiraHTTPClient struct {
    baseURL    string
    email      string
    token      string
    httpClient *http.Client
}

// NewJiraClient creates a JiraHTTPClient for the given Jira instance.
// baseURL must be the Atlassian Cloud URL (e.g., "https://mycompany.atlassian.net").
// email and token are the Atlassian account credentials for Basic auth.
func NewJiraClient(baseURL, email, token string) *JiraHTTPClient {
    return &JiraHTTPClient{
        baseURL:    strings.TrimRight(baseURL, "/"),
        email:      email,
        token:      token,
        httpClient: &http.Client{Timeout: 10 * time.Second},
    }
}
```

**Private basicAuthHeader helper:**
```go
func basicAuthHeader(email, token string) string {
    return "Basic " + base64.StdEncoding.EncodeToString([]byte(email+":"+token))
}
```

**ADF structs and helpers (private to package):**

Define ADF types for both reading and writing. Use `json.RawMessage` for recursive content to support arbitrary nesting:
```go
// adfNode represents a node in Atlassian Document Format content tree.
// Content is json.RawMessage to support recursive unmarshalling without
// requiring a fixed-depth struct hierarchy.
type adfNode struct {
    Type    string            `json:"type"`
    Text    string            `json:"text,omitempty"`
    Content []json.RawMessage `json:"content,omitempty"`
}

// adfDoc is the top-level ADF document envelope.
type adfDoc struct {
    Version int        `json:"version"`
    Type    string     `json:"type"`
    Content []adfNode  `json:"content"`
}
```

`extractADFText(raw json.RawMessage) string` — recursively unmarshal each node, collect text node values, join paragraphs with "\n\n" and list items with "\n". Handle node types: text, paragraph, hardBreak (append "\n"), bulletList/orderedList (recurse into children), listItem (recurse), codeBlock (collect text with code formatting). For unknown node types, recurse into Content if present.

`plainTextToADF(text string) adfDoc` — wrap in doc > paragraph > text:
```go
func plainTextToADF(text string) adfDoc {
    return adfDoc{
        Version: 1,
        Type:    "doc",
        Content: []adfNode{{
            Type: "paragraph",
            Content: []json.RawMessage{
                mustMarshal(adfNode{Type: "text", Text: text}),
            },
        }},
    }
}
```

**GetIssue implementation:**
- URL: `{baseURL}/rest/api/3/issue/{key}?fields=summary,description,status,priority,assignee,comment`
- `http.NewRequestWithContext(ctx, http.MethodGet, url, nil)`
- Set `Authorization: basicAuthHeader(...)` and `Accept: application/json`
- Execute request; handle errors from Do as `ErrJiraUnavailable` (wrapped)
- Map status codes: 401 → `ErrJiraUnauthorized`, 404 → `ErrJiraNotFound`, 429 → `ErrJiraUnavailable`, non-2xx → `ErrJiraUnavailable` with status code in message
- Decode response JSON into an anonymous struct with the Jira issue fields layout:
  ```
  fields.summary (string)
  fields.description (json.RawMessage — ADF)
  fields.status.name (string)
  fields.priority.name (string)
  fields.assignee.displayName (string)
  fields.comment.comments ([]struct{ author.displayName, body json.RawMessage, created string })
  ```
- Extract plain text from description and each comment body using `extractADFText`
- Parse comment CreatedAt using `time.Parse(time.RFC3339, created)` — use zero time on parse error (don't fail)
- Return `model.JiraIssue`

**AddComment implementation:**
- URL: `{baseURL}/rest/api/3/issue/{key}/comment`
- Build `commentRequest{ Body: plainTextToADF(body) }` and marshal to JSON
- `http.NewRequestWithContext(ctx, http.MethodPost, url, bytes.NewReader(payload))`
- Set `Authorization`, `Content-Type: application/json`, `Accept: application/json`
- Map 400 → `fmt.Errorf("jira: invalid comment body: %w", ErrJiraUnavailable)`, 401 → `ErrJiraUnauthorized`, 404 → `ErrJiraNotFound`
- Return nil on 201

**Ping implementation:**
- URL: `{baseURL}/rest/api/3/myself`
- GET with Authorization header only
- 200 → nil, 401 → `ErrJiraUnauthorized`, any other error/non-2xx → `ErrJiraUnavailable`
