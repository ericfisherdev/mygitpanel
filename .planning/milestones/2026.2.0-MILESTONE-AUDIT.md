## Tech Debt

### Phase 07 — Dockerfile Tailwind CLI Incompatibility

**Severity:** Deployment blocker (local development unaffected)

Tailwind standalone CLI binary (`tailwindcss-linux-x64`) uses glibc and fails on Alpine musl with `Error relocating /usr/local/bin/tailwindcss`. The `output.css` exists locally as a committed file, so `go run` works. Docker build stage fails.

**Workarounds:**
1. Use `tailwindcss-linux-x64` (glibc binary) with `apk add libc6-compat gcompat` in Alpine build stage
2. Switch build stage to `golang:1.25-bookworm` (Debian)
3. Pre-generate `output.css` locally, commit it, remove Tailwind step from Dockerfile
